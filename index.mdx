TizenBrew is a modular platform for Samsung Smart TVs (Tizen 3.0+) that lets you install and run custom apps and website modifications with a single click. No fighting with Tizen Studio every time you want to update an app.

## How it works

TizenBrew runs as a Tizen widget on your TV. A background Node.js service manages modules — fetching them from npm or GitHub, serving their files locally, and injecting scripts into websites. The UI lets you browse and launch your installed modules without ever touching Tizen Studio again.

<Steps>
  <Step title="Enable Developer Mode on your TV">
    Open **Apps**, enter `12345`, turn on Developer Mode, and enter your PC's IP address.
  </Step>
  <Step title="Install TizenBrew">
    Download and run [TizenBrew Installer](https://github.com/reisxd/TizenBrewInstaller/releases/latest) on your PC, then click **Install TizenBrew**.
  </Step>
  <Step title="Add modules">
    Launch TizenBrew on your TV, open **Module Manager**, and add modules by npm package name or GitHub repository.
  </Step>
  <Step title="Launch your app">
    Select any installed module from the home screen and it launches instantly.
  </Step>
</Steps>

## Key features

<CardGroup cols={2}>
  <Card title="One-click installs" icon="circle-check">
    Install apps and site mods directly from the TV UI — no PC commands needed after initial setup.
  </Card>
  <Card title="npm & GitHub modules" icon="box">
    Any npm package or GitHub repo can be a TizenBrew module, served via jsDelivr CDN.
  </Card>
  <Card title="Site modifications" icon="pen-to-square">
    Inject JavaScript into TV websites (like YouTube) to enable custom features.
  </Card>
  <Card title="Background services" icon="server">
    Modules can include a Node.js service that runs in the background, even when the module isn't active.
  </Card>
  <Card title="Auto-launch" icon="play">
    Configure a module to launch automatically when TizenBrew starts.
  </Card>
  <Card title="20+ languages" icon="language">
    TizenBrew's UI is localized into more than 20 languages.
  </Card>
</CardGroup>

<Note>
  TizenBrew requires a Samsung TV running Tizen 3.0 or newer (2017 models and later).
