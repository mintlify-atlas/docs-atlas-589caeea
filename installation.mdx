There are two supported ways to install TizenBrew. Both require a Samsung TV with Tizen 3.0 or newer and a PC on the same local network.

<CardGroup cols={2}>
  <Card title="TizenBrew Installer (Recommended)" icon="download" href="/installation/installer">
    A desktop app that installs TizenBrew for you with a single click. Works on every Samsung TV released since 2017. No command-line knowledge required.
  </Card>
  <Card title="Command line" icon="terminal" href="/installation/command-line">
    Install using Tizen Studio CLI tools. Use this method if you prefer manual control or already have Tizen Studio configured.
  </Card>
</CardGroup>

## Deprecated: USB Demo Package

<Warning>
  The USB Demo Package installation method is no longer available. Samsung shut down the service required to create USB Demo Packages, so this method cannot be used.
</Warning>

If you previously installed TizenBrew via USB Demo Package, use one of the two supported methods above to reinstall.
