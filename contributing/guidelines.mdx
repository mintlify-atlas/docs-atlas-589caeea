TizenBrew is an open-source project and welcomes contributions of all kinds. Whether you want to translate the UI, report a bug, build a module, or submit a code fix, there's a place for your work.

<Note>
  TizenBrew is licensed under the [GPL-3.0 license](https://github.com/reisxd/TizenBrew/blob/main/LICENSE). By contributing, you agree that your contributions will be distributed under the same license.
</Note>

## Ways to contribute

<CardGroup cols={2}>
  <Card title="Translations" icon="language" href="/contributing/translations">
    Add or improve a language translation. TizenBrew uses i18next with JSON language files — no build tooling knowledge required.
  </Card>
  <Card title="Bug reports" icon="bug" href="https://github.com/reisxd/TizenBrew/issues">
    Found something broken? Open an issue on GitHub with steps to reproduce, your TV model, and the Tizen version.
  </Card>
  <Card title="Module development" icon="puzzle" href="/modules/introduction">
    Build a module to extend TizenBrew with new apps or site modifications. Modules are standard npm packages with a few extra fields.
  </Card>
  <Card title="Code contributions" icon="code" href="https://github.com/reisxd/TizenBrew/pulls">
    Fix bugs or add features by submitting a pull request. Fork the repository, make your changes, and open a PR against the `main` branch.
  </Card>
</CardGroup>

## Development setup

To work on TizenBrew itself, you need to build and install the app from source. Follow the [rebuilding guide](/advanced/rebuilding) for step-by-step instructions on setting up your environment and deploying a development build to your TV.
