---
title: "User agent settings"
description: "Override the browser identification string TizenBrew sends to websites."
---

The user agent is a string that your browser sends to every website identifying itself — including the browser name, version, and operating system. Some websites use this string to decide which version of their content to serve.

TizenBrew lets you replace the default user agent with a preset string from a known Samsung TV model, or generate one based on your device's own firmware and model information.

<Tip>
  If a website or streaming service is showing an error, serving a degraded experience, or refusing to play content, try switching to a different user agent. Matching the user agent to your TV's Tizen version often resolves compatibility issues.
</Tip>

## Available presets

<AccordionGroup>
  <Accordion title="UE50MU7000 (Tizen 3)">
    A user agent string from a Samsung UE50MU7000 running Tizen 3.0. Use this on older TVs or when you need a Tizen 3 compatible identifier.

    ```text
    Mozilla/5.0 (LINUX; Tizen/3.0/2017.1.0) Cobalt/9.lts-gold (unlike Gecko) gles Evergreen/1.0.0 Starboard/9, Samsung_TV_KANTM_2017/T-MDEUC-1420.0 (Samsung, UE50MU7000, Wired)
    ```
  </Accordion>
  <Accordion title="QN55Q80AAFXZA (Tizen 6)">
    A user agent string from a Samsung QN55Q80AAFXZA running Tizen 6.0. Use this on newer TVs or when a Tizen 6 identifier is required.

    ```text
    Mozilla/5.0 (LINUX; Tizen/6.0/2021.1.3) Cobalt/21.lts.4.302899-gold (unlike Gecko) v8/7.7.299.8-jit gles Evergreen/1.4.3 Starboard/12, Samsung_TV_NIKEM2_2021/T-NKM2AKUC-2111.1 (Samsung, QN55Q80AAFXZA, Wired)
    ```
  </Accordion>
  <Accordion title="Based on your device">
    Generates a user agent string derived from your TV's actual firmware version and model name. TizenBrew reads this information from the Tizen system API and the local device API at `http://127.0.0.1:8001/api/v2/`, then appends it to the current browser user agent.
  </Accordion>
  <Accordion title="Default">
    Removes any custom user agent override, restoring the browser's built-in user agent string.
  </Accordion>
</AccordionGroup>

## Applying a user agent

Select a preset from the user agent settings screen. TizenBrew asks you to confirm the change and warns that it may have negative effects on some websites. After confirming, the user agent is saved to local storage and TizenBrew exits so the new value takes effect on the next launch.
